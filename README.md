# 🚀 Portfólio - Rafael Navarro

## 📌 Sobre o Projeto

Este projeto foi desenvolvido com o objetivo de apresentar minha trajetória como Desenvolvedor Front-End, destacando minhas habilidades técnicas, projetos e evolução profissional.

Após um período afastado da programação, retornei aos estudos e ao desenvolvimento de aplicações web, buscando aprimorar meus conhecimentos e construir soluções modernas, responsivas e focadas na experiência do usuário.

## 📂 Projetos Destacados

### 🧮 Calculadora React

Aplicação desenvolvida em React para realizar operações matemáticas básicas utilizando componentes reutilizáveis e gerenciamento de estado.

### 🌎 ODS

Projeto inspirado nos Objetivos de Desenvolvimento Sustentável da ONU, com foco em conscientização e apresentação visual acessível.

### ⚖️ Pearson Hardman

Landing page inspirada no universo da série Suits, desenvolvida com foco em design responsivo e experiência visual moderna.

### 🚚 Portflow

Projeto de interface para empresa de logística, com foco em organização visual, responsividade e boas práticas de desenvolvimento.

### 🎮 League of Legends

Projeto temático inspirado no universo League of Legends, explorando componentização, estilização e experiência do usuário.
